Layer Creation Options

Note: Layer Creation Options work with all layer types (Text, Shape, Null, Adjustment, Solid, Light, and Camera layers).

Version: 3.1.0

Help and Documentation

Features

Layer Creation Tools:

  • Text Layer - Creates a new text layer
  • Solid Layer - Creates a new solid layer
  • Light - Creates a new light
  • Camera - Creates a new camera
  • Null Object - Creates a new null object
  • Shape Layer - Creates a new shape layer
  • Adjustment Layer - Creates a new adjustment layer

Editing Tools:

  • Delete - Removes selected layers
  • Sequence - Creates a sequence from selected layers
  • Split - Splits layers at current time
  • Precompose - Precomposes selected layers
  • New Composition - Creates a new composition

Settings & Options:

  • Auto Parent Layers - Automatically parents selected layers
  • Match Layer Duration - Matches duration of new layers
  • Auto Delete Split Parts - Automatically deletes split parts
  • Show Delete Confirmation - Shows confirmation before deleting

Note: Auto Parent and Match Duration options now work with all layer types.
